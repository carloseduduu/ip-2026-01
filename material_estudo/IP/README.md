**📂 Organização do Repositório**  
Para facilitar seu estudo, o conteúdo está dividido em cinco trilhas principais:  
**1\. Trilha: Bases da Lógica e Algoritmos**  
Foco no pensamento computacional e ferramentas didáticas.

* **Conteúdo:** Conceitos de variáveis, tipos de dados (inteiro, real, literal, lógico) e a ferramenta **VisuAlg** para praticar com **Portugol**.  
* **Estruturas:** Seleção (se-então) e Repetição (para, enquanto).

**2\. Trilha: Dominando a Linguagem Go**  
Transição da lógica para uma linguagem de alta performance utilizada pelo Google.

* **Conteúdo:** Sintaxe básica, pacotes (`fmt`), funções, arrays e slices.  
* **Compilação:** Uso dos comandos `go build` e `go run` para executar seus programas.

**3\. Trilha: Algoritmos de Busca e Ordenação**  
Aprenda a organizar e encontrar dados de forma eficiente.

* **Ordenação:** Pratique algoritmos como Bubble Sort, Selection Sort e Quick Sort.  
* **Busca:** Entenda a diferença de performance entre a busca sequencial e a busca binária.  
* **Análise:** Introdução à **Notação Big O** para medir o desempenho do seu código.

**4\. Trilha: Desenvolvimento Web com Go**  
Transforme seu código em um serviço acessível pelo navegador.

* **Frontend:** Estruturação com **HTML** e estilização com **CSS**.  
* **Backend:** Criação de servidores HTTP, gerenciamento de rotas e tratamento de arquivos estáticos em Go.

**5\. Trilha: Persistência e Infraestrutura**  
Aprenda a salvar dados e preparar sua aplicação para o mercado.

* **Banco de Dados:** Linguagem SQL e uso do **PostgreSQL** para armazenar informações de usuários.  
* **Docker:** Uso de containers para isolar o ambiente do banco de dados e facilitar a configuração.
